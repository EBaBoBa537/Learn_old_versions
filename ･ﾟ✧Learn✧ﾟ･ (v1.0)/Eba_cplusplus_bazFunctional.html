Структура программы

Код состоит из стейтментов — структурных единиц кода, которые обязательно заканчиваются точкой с запятой. Пример стейтмента: int value = 15; . Стейтменты могут объединяться в блоки с помощью составного оператора {}.




Пример №1:   HelloWorld
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!";
    return 0;
}
 



Директивы препроцессора

Директивы препроцессора — специальные функции, обрабатываемые препроцессором.

⬥ #include — сообщение препроцессору искать файл в системных путях либо в текущей директории.
⬥ #define — определение фрагмента кода
⬥ #undef — отменить определение фрагмента кода
⬥ #ifdef – #endif — если фрагмент кода определён
⬥ #ifndef – #endif — если фрагмент кода не определён
⬥ #if – #elif – #else — если выражение верно – иначе если выражение верно – иначе
⬥ #line — изменить номер текущей строки и имя файла
⬥ #error — выдать пользовательское сообщение об ошибке
⬥ #pragma once — подключить только один раз
⬥ #warning — сообщение об ошибке без остановки компиляции
⬥ ...

Директива #include — сообщение препроцессору искать файл в системных путях либо в текущей директории. Она копирует текст из файла без проверок содержимого. Синтаксис:
#include <fileName>, если нужно найти файл fileName в системных путях (как правило, так подключаются стандартные библиотеки)
#include "fileName.h", если файл fileName в текущей директории (так можно подключить собственные библиотеки).

Пример №1:   #ifndef – #endif
#ifndef FUNCTION_H
#define FUNCTION_H

bool trueFunction() {
    return true;
}

#endif

Пример №2:   #pragma once
#pragma once

int function() {
    return 5;
}


Пример №3:   #include
(файл function.h из примера №1 и данный файл находятся в одной директории)
#include "function.h"

int main() {
    int trueValue = trueFunction();
    return true;
}
/* Теперь значение trueValue равно true */

 



Типы данных

Тип данных — одно из двух свойств выражений в С++. Бывают типы:
⬥ bool — логический тип. Может принимать значения true или false
⬥ int — целочисленный тип. Примеры значений: 0, -12, 1, и др.
⬥ float — тип чисел с плавающей точкой. Примеры значений: 1.23, -0.3333, 42.12, и др.
⬥ double — тип чисел с плавающей точкой большего размера, чем float. Примеры значений: 423.1, -8.0, 124.23, и др.
⬥ char — символьный тип. Примеры значений: 'a', '4', '-', '\n', и др.

Операторы и перегрузка операторов

Операторы сообщают компилятору о необходимости выполнения какого-либо математического (с использованием операндов) или логического действия.

⬥ {} — составной оператор
⬥ () — оператор вызова / скобки в выражении / инициализация
⬥ , — оператор–разделитель выражений
⬥ ; — оператор–разделитель стейтментов
⬥    (пробел) — ...
⬥ \n (переход на новую строку) — ...
⬥ :: — оператор области видимости
⬥ ?: — условный тернарный оператор

Операторы доступа:
⬥ . — оператор доступа к члену объекта
⬥ [] — индекс массива

⬥ –> — доступ к члену объекта через указатель объекта
⬥ –>* — доступ к члену объекта через указатель на член объекта
⬥ .* — доступ к члену объекта через указатель на член объекта
Математические операторы:
⬥ + — сложение / унарный плюс
⬥ – — вычитание / унарный минус
⬥ * — умножение
⬥ / — деление
⬥ % — остаток от деления

Оператор присваивания:
⬥ = — присваивание

Арифметические операторы присваивания:
⬥ += — присваивание со сложением
⬥ –= — присваивание с вычитанием
⬥ *= — присваивание с умножением
⬥ /= — присваивание с делением
⬥ %= — присваивание с остатком от деления

Операторы сравнения:
⬥ == — равно
⬥ != — не равно
⬥ > — больше
⬥ >= — больше или равно
⬥ < — меньше
⬥ <= — меньше или равно

Логические операторы:
⬥ ! — НЕ
⬥ && — И
⬥ || — ИЛИ

Побитовые операторы:
⬥ ~ — побитовое НЕ
⬥ & — побитовое И
⬥ | — побитовое ИЛИ
⬥ ^ — побитовое исключающее ИЛИ
⬥ << — побитовый сдвиг влево
⬥ >> — побитовый сдвиг вправо
⬥ &= — присваивание с побитовым И
⬥ |= — присваивание с побитовым ИЛИ
⬥ ^= — присваивание с побитовым исключающее ИЛИ
⬥ <<= — присваивание с побитовым сдвигом влево
⬥ >>= — присваивание с побитовый сдвигом вправо
Операторы инкремента–декремента:
⬥ ++ — пост-инкремент / пре-инкремент
⬥ –– — пост-декремент / пре-декремент
Оператор адреса:
⬥ & — адрес
Оператор разыменования:
⬥ * — разыменование

Переменные, указатели, ссылки

Переменная — это объект с именем, которых хранит значение l-value.
Правила называния переменных:
– Имена переменных могут содержать буквы A–Z и a–z, цифры 0–9 и знак подчеркивания "_"
– Имя не может начинаться с цифры
– Регистр учитывается, то есть NAME и name — это разные имена

Перечисления

Перечисления — это тип данных, объекты которого хранят перечислители. Перечислители — это нумерованные символьные переменные. Поскольку перечислители вместе с перечислением находятся в едином пространстве имен, то имена перечислителей не могут повторяться в разных перечислениях.
enum Numbers {}

Пример №1:   Создание перечисления
enum Numbers {
   ONE,
   TWO,
   THREE = 3,
}
