I уровень: Системы счисления

Система счисления — ........... Существует бесконечное множество систем счислений. Наиболее используемые: десятичная (DEC), восьмеричная (OCT), шестнадцатеричная (HEX), и двоичная (BIN)
Двоичная система счисления BIN — позиционная система счисления с основанием 2.

Например, число 2753 в DEC представляется так:
2753 = 2*103 + 7*102 + 5*101 + 3*100

В BIN вот так:
101011000001 = 1*211 + 0*210 + 1*29 + 0*28 + 1*27 + 1*26 + 0*25 + 0*24 + 0*23 + 0*22 + 0*21 + 1*20 = 2753

А в HEX вот так:
AC1 = A*162 + C*161 + 1*160 = 10*162 + 12*161 + 1*160 = 2753

II уровень: Данные

Бит — минимальная единица информации, которая может принимать значения 0 или 1.

Байт — минимальная ячейка адресации — единица информации, содержащая 8 бит, которая может принимать значения от 0 до 255.




С двоичными числами, записанными в битах, можно выполнять математические операции, операции сравнения и перевода в другой вид.

Вычитание в большинстве реализаций осуществляется с помощью дополнительного кода — способа записи отрицательных чисел, при котором вычитание реализуется так же, как и сложение.



Прямой код — модуль числа со старшим знаковым битом (0 — положительное число, 1 — отрицательное).
Обратный код — для положительных чисел совпадает с прямым кодом, для отрицательных — инвертированный прямой код.
Дополнительный код — для положительных чисел совпадает с прямым кодом, для отрицательных — сумма инвертированного прямого кода и 1.

Порядок байтов
big endian — от старшего к младшему (слева направо)
little endian — от младшего к старшему (обратный, справа налево)

IV уровень: Простейшие цифровые устройства


Комбинационные цифровые устройства — устройства ...
Последовательностные цифровые устройства — устройства ...
Преобразователь кодов — КЦУ, преобразующее один код в другой



Шифратор — КЦУ (преобразователь кодов), преобразующее n–разрядный позиционный код в m–разрядный двоичный



Дешифратор — КЦУ (преобразователь кодов), преобразующее n–разрядный двоичный код в m–разрядный позиционный



Мультиплексор — КЦУ, имеющее несколько сигнальных входов, один/несколько управляющих входов и один выход. Позволяет передать сигнал с одного из входов на выход, при этом выбор нужного входа осуществляется комбинацией управляющих сигналов



Демультиплексор — КЦУ, имеющее один сигнальный вход, один/несколько управляющих входов и несколько выходов. Позволяет сигнал с входа на один из выходов, при этом выбор нужного выхода осуществляется комбинацией управляющих сигналов



Полусумматор — КЦУ, предназначенные для сложения чисел



Сумматор — КЦУ, предназначенные для сложения чисел
Полные сумматоры можно собирать в каскады, с помощью которых можно складывать числа любой битности



Умножитель — КЦУ, предназначенные для умножения чисел

Вычитатель — КЦУ, предназначенные для вычитания чисел. Бывает вычитатель в прямых кодах и вычитатель в дополнительных кодах

Компаратор — КЦУ, предназначенные для сравнения чисел



Триггер — ПЦУ, предназначенные для записи и хранения информации

IX уровень: Операционная система

Операционная система ОС — ПО, управляющее компьютерным оборудованием и предоставляющее пользовательским программам техническую базу

Ядро ОС — (часть ОС, обеспечивающая основные низкоуровневые функции ОС, к которой пользователь не имеет доступа)

Процесс — ...

Поток / thread — элемент выполнения процесса — наименьшая единица обработки с точки зрения ОС

Блок управления процессов PCB — структура данных, поддерживаемая ОС для каждого процесса

Задачи ОС:
⬥ Загрузка пользовательских программ в оперативную память и их исполнение
⬥ Огранизация доступа к периферийным устройствам
⬥ Управление оперативной памятью, организация виртуальной памяти
⬥ Управление доступом к энергозависимым носителям памяти
⬥ Обеспечение пользовательского интерфейса
⬥ Огранизация многопоточности

Файловая система — .порядок организации, хранения и именования данных на носителях информации
