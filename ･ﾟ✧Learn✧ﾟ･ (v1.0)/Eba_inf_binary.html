Системы счисления

Система счисления — ........... Существует бесконечное множество систем счислений. Наиболее используемые: десятичная (DEC), восьмеричная (OCT), шестнадцатеричная (HEX), и двоичная (BIN)

Например, число 2753 в DEC представляется так:
2753 = 2*103 + 7*102 + 5*101 + 3*100

В BIN вот так:
101011000001 = 1*211 + 0*210 + 1*29 + 0*28 + 1*27 + 1*26 + 0*25 + 0*24 + 0*23 + 0*22 + 0*21 + 1*20 = 2753

А в HEX вот так:
AC1 = A*162 + C*161 + 1*160 = 10*162 + 12*161 + 1*160 = 2753

Единицы информации

Бит — минимальная единица информации, которая может принимать значения 0 или 1.


Байт — единица информации, содержащая 8 бит, которая может принимать значения от 0 до 255.
